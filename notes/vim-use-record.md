###vim

1. `esc` -- turn to command mode
1. `i` -- turn to edit mode and ready to insert in line
2. `o` -- turn to edit mode and ready to insert next line
3. `h` -- left
7. `j` -- down
8. `k` -- up
9. `l` -- right
3. `^` -- start in line
4. `$` -- end of line
5. `dw` -- delete word
6. `dd` -- delete line
7. `yy` -- copy line
8. `p` -- paste
9. `e` -- skip an word



###vim diff
1. `dp` -- switch left to right
2. `dd` -- delete one line
